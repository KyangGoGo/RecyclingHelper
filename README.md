실시간 이미지 인식을 통하여 재활용 쓰레기 품목별 분비배출방법을 안내하는 안드로이드 어플리케이션

## 사용 방법
### 재활용 쓰레기 인식
1. 인식할 재활용 쓰레기에 카메라를 둔다.
2. 앱 하단바에 정확도를 기준으로 세 가지 인식한 품목이 나타난다.
3. 하단바를 위로 올리면 랜덤 꿀팁이 나온다!

### 간단한 정보 보기
1. 앱 하단바의 인식한 품목버튼을 누른다.
2. 간단한 정보를 볼 수 있다.

### 자세한 정보 보기
1. 품목별 자세한 정보를 보여준다.
2. 간단한 정보 보기에서 자세히 보기 버튼을 누른다.
3. 자세한 품목별 배출 방법
4. 주의해야할 점
5. 자주 묻는 질문

## 현재 분류 가능한 재활용품

2. 스티로폼
3. 유리병
4. 종이
5. 캔류
6. 폐전지
